제 1 장. 시스템 시작하기

1.1 시스템 접속 및 로그인

웹 브라우저를 통해 LOGI1 ERP 시스템에 접속합니다. 본 시스템은 보안 문(Access Gate)을 통해 인가된 사용자만 접근할 수 있습니다.

그림 1-1. Access Gate 로그인 화면

로그인 절차

  1. 발급받은 가입 코드와 구글 계정을 입력합니다.
  2. 보안을 위해 2단계 인증이 설정된 경우, 모바일 기기로 본인 인증을 합니다.
  3. [로그인] 버튼을 클릭하여 시스템에 진입합니다.
※ 참고: ERP 첫 사용 시, 최초 로그인된 구글 계정 이름으로 프로필 설정이 됩니다. 로그인 전, 본인의 구글 계정 프로필 이름을 확인하세요.

제 2 장. 메인 대시보드

2.1 대시보드 구성요소

로그인 후 가장 먼저 나타나는 화면으로, 사용자의 주요 업무 현황과 각종 연동 데이터(날씨 및 선박 스케쥴)를 한눈에 파악할 수 있도록 구성되어 있습니다.

그림 2-1. 메인 대시보드 화면 구성

주요 위젯 설명

위젯명 설명 및 기능
업무 요약 입항 및 작업 예정 선박 건수, 등록된 B/L 건수 등 핵심 업무 지표를 숫자로 표시합니다.
캘린더 선박 입출항 일정 등을 달력 형태로 제공합니다. (추후 '팀 캘린더 보기'를 도입하여 주요 결제일 및 미팅 일정 등도 추가 가능케 수정 예정)
날씨 정보 부산 앞바다의 해상 기상 정보를 실시간으로 연동하여 보여줍니다. (선원 교대 및 급유 등, 선박 작업 스케쥴 수립 시 참고)
시스템 알림 신규 메시지, 화물 내역 수정 등 알림이 좌측 하단 메신저 앱 및 상단 종 아이콘에 빨간 점 형태로 표시됩니다.

제 3 장. 주요 물류 업무 관리

3.1 선박 목록

입항 예정 및 입항된 선박의 상세 정보와 현재 상태를 조회하고 관리하는 메뉴입니다. (메뉴 경로: 좌측 사이드바 > 선박 관리)

선박 목록 조회

등록된 전체 선박의 리스트가 표시되며, 선박명, 항차, ETA 및 ETD, 운항 상태 등을 카탈로그 형태로 확인할 수 있습니다. 상단의 검색 바를 통해 특정 선박을 빠르게 찾을 수 있습니다.

선박 상세 정보 (개발 중)

추후 추가 예정입니다. 특정 선박을 클릭하면 상세 페이지로 이동합니다. 이곳에서는 이곳에서는 선박의 제원, 과거 운항 이력을 조회할 수 있을 예정입니다.

3.2 화물 관리 (Cargo Management)

위탁받은 수출입 화물의 정보 등록부터 D/O까지의 전 과정을 추적 관리합니다.

그림 3-1. 신규 화물 등록(Register Cargo) 팝업 화면

신규 화물 등록

  1. 상단의 [신규 화물 등록] 버튼을 클릭합니다.
  2. 화주 정보, 품목명, HS Code, 중량(Weight), 용적(CBM), 포장 형태를 정확히 입력합니다.
  3. 또한 AI 기능을 이용한 OCR 자동 분석 기능을 이용해 등록가능합니다.
  4. 직접 입력의 경우 [저장]을 눌러 저장하거나, OCR 분석의 경우 자동으로 저장됩니다.

3.3 선적 및 B/L 관리

선적 상세 내역

특정 선박에 할당된 화물들의 적재 계획(Stowage Plan)과 선적 진행 상태를 확인합니다. 체크리스트(CheckList) 기능을 통해 누락된 절차가 없는지 점검할 수 있습니다.

선하증권 및 기타 문서 관리

B/L(선하증권) 및 기타 문서들을 하고 관리합니다. B/L 및 기타 제반 서류를 클라우드로 어디서든 조회할 수 있으며, 다운로드 가능합니다.

제 4 장. 커뮤니케이션 및 파일 관리

4.1 사내 메신저 (Chat Window)

ERP 시스템 내부에 통합된 실시간 메신저입니다. 별도의 프로그램 설치 없이 업무 창을 띄운 상태로 타 부서 담당자(예: 회계팀 ↔ 운항팀)와 소통할 수 있습니다.

  • 다양한 활용: 대화 메세지 버블에 '반응'을 남기거나, 답장을 하는 등 다양한 의사소통이 가능합니다.
  • 업무 링크: 특정 화물 번호나 선박 번호를 입력하면 클릭 가능한 링크로 자동 변환될 예정입니다(개발 중).

4.2 클라우드 파일함 (Cloud File Manager)

업무와 관련된 모든 서류(B/L 사본, 인보이스, 패킹리스트, 통관 서류, 기타 청구서 등)를 중앙 집중식으로 보관하는 클라우드 스토리지 공간입니다.

그림 4-1. 클라우드 파일 관리자 폴더 구조 화면

주요 기능

  • 글로벌 클라우드 관리: 선박별로 관리되는 문서 및 파일들을 전체 열람가능합니다. (Global Cloud Manager).
  • 폴더 구조: 연도별, 화주별, 선박별로 자동 생성된 폴더 트리를 제공할 예정입니다.(개발 중)

제 5 장. 업무 지원 도구 (Toolbox)

효율적인 업무 처리를 위해 제공되는 부가 기능 모음입니다. 우측 상단의 '도구함(Toolbox)' 아이콘을 클릭하여 팝업 창 형태로 실행할 수 있습니다.

그림 5-1. 통합 툴박스(Toolbox) 팝업 실행 화면

5.1 HS Code 검색기 (HS Code Search)

수출입 통관 시 필수적인 품목분류코드(HS Code)를 시스템 내에서 즉시 검색할 수 있습니다. 품목명이나 키워드를 입력하면 관련 코드 및 복사가 가능합니다.

5.2 개인 메모장 (My Memo)

포스트잇을 대체하는 디지털 메모장입니다. 업무 중 떠오른 아이디어나 임시로 복사해 둔 텍스트, 송장 번호 등을 간편하게 기록하고 저장할 수 있습니다. 이 데이터는 사용자 본인에게만 노출됩니다.

5.3 팀 게시판 (Team Board)

부서 내 공지사항, 업무 인수인계, 휴가 일정 등을 공유하는 게시판입니다. 리치 텍스트 에디터(Rich Editor)를 지원하여 표, 이미지, 글머리 기호 등을 활용한 깔끔한 문서 작성이 가능합니다.
